## Para gerar a API KEY:

### Acesse o Site
Vá em console.groq.com e clique em "Sign Up" para iniciar o processo de registro.

### Crie Sua Conta
Use sua conta GitHub, Google ou e-mail. Confirme seu e-mail se for solicitado para
ativar sua conta.

### Gere a Key
No dashboard, navegue até "API Keys" e selecione "Create API Key". Dê um nome à
sua chave (ex: "Chatbot Aula") e clique em "Create Key".

### Configuração da API
No código, substitua:

api_key = 'SUA CHAVE'

# Minha_IA

Um chatbot simples desenvolvido em Python utilizando:

- LangChain
- Groq API
- Modelo Llama 3.3 70B
- Prompt Templates

O projeto simula uma conversa contínua com histórico de mensagens armazenado durante a execução.

---

# Tecnologias Utilizadas

- Python 3.10+
- LangChain
- LangChain Groq
- Llama 3.3 70B Versatile
